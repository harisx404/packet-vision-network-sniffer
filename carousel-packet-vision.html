@harisx404 Memory Safety

Defeating OOM Errors

FIFO Sliding-Window Buffer

Infinite packet captures lead to infinite RAM consumption. I implemented a strict FIFO sliding-window buffer — when the buffer reaches 50,000 packets, the oldest entry is evicted before the newest is appended. Result? Zero Memory Leaks.

src/logger.py
01 def log_packet(self, packet_info, raw_packet=None) -> None:
02     if raw_packet is not None:  # CRITICAL RAM PROTECTION
03         if len(self._raw_packets) >= self.config.max_packets:
04             self._raw_packets.pop(0)  # Remove oldest (FIFO method)
05         self._raw_packets.append(raw_packet)
